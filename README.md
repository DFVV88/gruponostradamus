# Grupo Nostradamus

Sitio web institucional estático del Grupo de Estudio Nostradamus.

## Estructura principal

- `index.html`: página principal.
- `clases-en-vivo.html`: página interna de acceso guiado a Microsoft Teams y soporte.
- `ciclos.html`: oferta académica.
- `ciclo-*.html`: páginas individuales de ciclos.
- `blog*.html`: noticias y artículos.
- `assets/css/`: estilos del sitio.
- `assets/js/`: scripts del sitio.
- `assets/img/`: imágenes y recursos gráficos.
- `sitemap.xml`: mapa del sitio para buscadores.
- `robots.txt`: reglas de rastreo e indexación.
- `_headers`: cabeceras de seguridad y caché para hosting compatible.

## Mantenimiento recomendado

- Usar siempre enlaces seguros `https://`.
- Mantener una sola página principal: `index.html`.
- Optimizar imágenes antes de subirlas, idealmente en formato WebP cuando sea posible.
- No subir contraseñas, claves privadas, tokens, credenciales ni archivos `.env`.
- Revisar `sitemap.xml` cuando se agreguen o retiren páginas.
- Validar que los botones comerciales principales apunten a WhatsApp o a páginas internas claras.
- Mantener los metadatos SEO importantes directamente en el `<head>` de las páginas principales.
- Revisar la versión móvil antes de publicar cambios.

## Páginas prioritarias para SEO

- Inicio: `https://gruponostradamus.edu.pe/`
- Clases en vivo: `https://gruponostradamus.edu.pe/clases-en-vivo.html`
- Ciclos académicos: `https://gruponostradamus.edu.pe/ciclos.html`
- Ciclos UNI individuales: `ciclo-anual-uni.html`, `ciclo-semianual-uni.html`, `ciclo-semestral-uni.html`, `ciclo-elite-uni.html`, `ciclo-repaso-uni.html`, `ciclo-paralelo-cepre-uni.html`
- Contacto: `contacto.html`

## Checklist antes de publicar

1. Revisar que no existan enlaces rotos.
2. Confirmar que WhatsApp abra con el mensaje correcto.
3. Probar menú móvil, botones y formularios.
4. Verificar que `sitemap.xml` incluya páginas nuevas.
5. Comprimir imágenes nuevas antes de subirlas.
6. Confirmar que no se suban datos sensibles.
