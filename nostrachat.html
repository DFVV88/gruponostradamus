Uso responsable

Reglas para cuidar la comunidad

NostraCHAT no es un chat libre sin control. Es un espacio académico con trazabilidad, reglas y moderación.

1. Solo uso académico

Las consultas deben estar relacionadas con clases, ejercicios, ciclos, orientación o preparación UNI.

2. Respeto obligatorio

No se permiten insultos, burlas, spam, discusiones ofensivas ni mensajes que afecten a otros alumnos.

3. Imágenes controladas

La subida de fotos se habilitará más adelante. Solo se permitirán imágenes de ejercicios o problemas académicos.

4. Moderación activa

Los moderadores podrán borrar mensajes, retirar imágenes, reportar usuarios o suspender accesos.

5. Sin datos personales

No se deben publicar documentos, datos privados, capturas personales ni información sensible de terceros.

6. Acceso identificable

Los alumnos ingresan con datos verificables. Los externos deben registrar nombre y WhatsApp o motivo de consulta.

Ruta de implementación

Desarrollo por etapas

Para evitar gastos innecesarios y controlar el uso, NostraCHAT se implementa paso a paso.

1

Diseño oficial

Página visual, zonas, reglas y estructura propia dentro de la web.

2

Chat de texto

Mensajes en tiempo real con salas separadas para alumnos y externos.

3

Control de usuarios

Registro básico, identificación de alumnos y acceso controlado.

4

Fotos moderadas

Subida limitada de imágenes académicas con reglas de seguridad.
